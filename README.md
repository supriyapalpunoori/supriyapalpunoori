### 🏆 Featured Projects

#### 😴 AI-Based Sleep Detection System

> **Python · Machine Learning · Random Forest · Time Series Analysis**

Developed a machine learning system to detect sleep onset and wake-up events using wearable accelerometer sensor data. Performed feature engineering on ENMO and z-angle signals and compared multiple models, achieving the best performance with Random Forest.

---

#### 💇 Hair Health Prediction System

> **Python · Scikit-Learn · Streamlit · Machine Learning**

Built a classification model to predict hair health status as Healthy, Moderate, or Unhealthy. Applied advanced preprocessing techniques including OneHotEncoder and OrdinalEncoder and achieved 98% accuracy using Decision Tree. Deployed the application using Streamlit.

---

#### 🤖 Document-Based RAG Chatbot

> **LangChain · FAISS · Gemini API · Streamlit**

Developed a Retrieval-Augmented Generation chatbot capable of answering questions from PDF documents. Implemented document chunking, vector embeddings, semantic search, and context-aware response generation.

---

#### 📊 Consumer Behaviour Analysis of Beauty Products

Performed Exploratory Data Analysis to uncover customer purchasing patterns, product preferences, and market trends. Generated visual insights and recommendations for data-driven decision-making.
